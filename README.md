# M.log

개발하면서 markdown으로 정리를 하고 필요시 검색할 수 있도록 정리

## install

```cmd
pnpm install
```

## Getting Started

```cmd
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## markdown 

`src/contents` 에 위치.

상단에 필요정보 노출
```markdown
---
title: title
update: date
tags: "tag, tag"
---
```
